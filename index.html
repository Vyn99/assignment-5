How to apply CSS to HTML

To style HTML using external CSS, use link, i.e: 

<link rel="stylesheet" href="styles.css">

To link to an external JavaScript file, you must use script with a source i.e: 

<script src="path/to/my/script.js"></script>
To write JavaScript code in a HTML file, use script without a source i.e: 
<script>
            window.addEventListener("load", () => {
                console.log("This function is executed once the page is fully loaded");
                });
            </script>
